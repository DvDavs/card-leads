# Prompt: propuesta de sistemas por rubro

> BORRADOR. Aun no implementado.

Dado el lead (rubro + servicios), genera `proposal.md` con 2-4 ideas de
sistemas/soluciones concretas para ese negocio. Usar como semilla las
`proposalIdeas` de config/rubro-map.ts.

Formato por idea:
- **Nombre** — que resuelve, en una frase.
- Beneficio directo para el negocio.
- Esfuerzo estimado (bajo / medio / alto).

Reglas:
- Concreto y accionable. Nada de humo.
- Priorizar lo que impacte captar/retener clientes.
